# Visual Studio Code 存檔自動排版

## 1. 開啟 setting (Ctrl + ,)

![setting](https://i.imgur.com/AGKHwWU.png)

## 2. 輸入 "Format On Save" > 將它打勾 ， 這樣存檔的時候就會自動排了

![Format](https://i.imgur.com/uSnUZl7.png)
